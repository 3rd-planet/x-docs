you can use the following services to deploy your application:

- [forever](https://www.npmjs.com/package/forever) - a simple CLI tool for ensuring that a given script runs continuously (i.e. forever)
- [pm2](https://www.npmjs.com/package/pm2) - a production process manager for Node.js applications with a built-in load balancer
- [nodemon](https://www.npmjs.com/package/nodemon) - a utility that will monitor for any changes in your source and automatically restart your server
- [supervisor](https://www.npmjs.com/package/supervisor) - a supervisor program for running nodejs programs

> **Note:** The entry point of your application is `server.js` file **NOT `index.js` file.**
